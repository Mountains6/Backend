### Local storage vs cookie

LocalStorage, cookies - оба хранилища браузерные

Cookies - автоматически добавляются в запрос

Cookies - можно хранить токен (задаём срок хранения)
LocalStorage - theme - client settings (данные храняться, пока мы не решаем их удалить)
SessionStorage - inputs (данные храняться пока не закроется вкладка)
